# Personal Portfolio Website

This project is a responsive personal portfolio website created as part of a web development challenge.  
It showcases my profile, skills, and projects, and serves as an online portfolio to demonstrate my web development abilities.

The website is designed with a clean and organized layout, smooth navigation, and responsiveness across mobile and desktop devices.
